# 旅游行业管理与运行调度系统 - 项目总结

## 项目简介

**旅游行业管理与运行调度系统**是一个基于Spring Boot的综合性旅游管理平台，实现了对旅游景区、饭店、旅行社、导游、车辆等旅游要素的全方位监控和管理，提供告警管理、投诉处理、诚信档案等完整功能。

---

## 核心功能总结

### ✅ 已实现功能

#### 1. 景区管理
- 景区基础信息（容量、停车位）
- 实时游客数量监控
- 管理人员和应急救援信息
- 地理位置和电子地图

#### 2. 饭店管理
- 星级评定和接待能力
- 实时入住率监控
- 停车位管理

#### 3. 旅行社及车辆管理
- 旅行社资质和诚信管理
- 车辆GPS实时追踪
- 电子行程单和轨迹偏离检测
- 出境游签证审核

#### 4. 导游管理
- 注册登记和资格审验
- 求职应聘管理
- 实时位置和团队监控

#### 5. 告警管理
- 多类型告警规则设定（容量、停车、入住率、GPS偏离等）
- 告警报告生成和处理
- 分级告警机制（提示/警告/严重）

#### 6. 投诉管理
- 投诉记录和处理流程
- 首问负责制
- 满意度评价

#### 7. 诚信档案
- 企业诚信评价体系
- 投诉关联和统计
- 诚信等级管理

## 技术架构

### 后端
- **框架**: Spring Boot 4.0.1
- **ORM**: Spring Data JPA + Hibernate
- **数据库**: MySQL 5.7+
- **工具**: Lombok

### 前端
- **技术**: HTML5 + CSS3 + JavaScript + jQuery
- **地图**: 百度地图集成

### 架构模式
- **分层架构**: Controller → Service → Dao → Entity
- **RESTful API**: 标准HTTP方法
- **前后端分离**: 前端通过AJAX调用后端API
